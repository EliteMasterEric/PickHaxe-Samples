## Notes

- To manually run a workflow:
    - Navigate to Actions
    - Select the workflow you want to run.
    - Click "Run workflow" at the right side
- `git add .; git commit -m "Workflow"; git push`
- https://github.com/nektos/act#manual-download
    - `./bin/act -j build`
